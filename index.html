HTML/CSS CheatSheet

Table elements

Tag Name Description
<table> Table The wrapper for HTML tables
<tr> Table Row Add row to table
<td> Table Data Add cells of data to table
<thead> Table Head Heading of table columns
<tbody> Table Body Semantic - Contains table data other than heading and footer
<th> Table Heading Add titles for rows - must be enclosed in table row element
<tfoot> Table Footer Add footer to table - uses table rows

Table Attributes

Attribute Name Description
colspan Column Span Indicates how many columns a cell should span
rowspan Row Span Indicates how many rows a cell should span
